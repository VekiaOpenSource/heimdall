Welcome to Heimdall ssh replicator authentication key web page.

This project is a python made application to manage the server authentication on ssh service. Consider that the rsa key authentication as the most secure connection. It should be usefull for big infrastructure to connect without know any user passwords. That is the Heimdall philosophy.
